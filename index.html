Central de Ajuda & Manual Completo v6.5.0

📅 1. Navegação e Grade Diária

A tela principal mostra a grade de horários das salas. Cada coluna representa uma sala e cada linha um horário (das 7h às 21h).

  • Clique no símbolo + em um slot vazio para criar uma nova reserva.
  • Clique em uma reserva existente para editar seus detalhes.
  • Use o campo de busca para destacar os horários de um profissional específico.
  • O percentual de ocupação de cada sala aparece no cabeçalho da coluna.
  • Use as setas ← → ou o calendário para navegar entre datas.

📝 2. Criação de Reservas

Preencha os campos ao criar uma nova reserva:

  • Profissional: Selecione o fisioterapeuta responsável.
  • Paciente: Busque por celular (000-00000-0000) ou cadastre novo.
  • Sala: Escolha a sala desejada.
  • Status: Confirmado (conta para faturamento), Pendente, Finalizado ou Falta.
  • Horário: Defina início e término.
  • Recorrência: Marque a opção para agendar automaticamente nas próximas semanas (2 a 20 semanas), sempre no mesmo dia da semana.
  • Segundo Profissional: Disponível apenas na sala Pilates. Permite compartilhar a reserva com outro profissional.
  • Observações: Registre o histórico do atendimento (fica salvo no prontuário do paciente).
  • Autenticação: Informe sua senha pessoal ou a senha do Administrador para salvar.

🔐 3. Segurança e Permissões

  • Edição: Apenas o criador da reserva ou o Administrador podem editar (é solicitada a senha).
  • Exclusão: Apenas o criador ou o Administrador podem excluir. Todas as exclusões são registradas no log de auditoria com data, horário, quem excluiu e IP.
  • Senha Master: Necessária para cadastrar, editar ou excluir profissionais.
  • Senha Administrativa: Necessária para acessar o painel administrativo.
  • Cadastro de Pacientes: Apenas o profissional que cadastrou ou admin pode editar/excluir. Pacientes podem ter múltiplos profissionais vinculados.

👥 4. Módulo de Pacientes

Ao criar uma reserva, você pode:

  • Buscar um paciente cadastrado pelo celular (formato: 000-00000-0000).
  • Se encontrar, o nome é preenchido automaticamente (sempre em MAIÚSCULAS).
  • Se não encontrar, cadastre o paciente rapidamente sem sair da tela.
  • O sistema pergunta se o paciente também é atendido por outros profissionais.
  • O campo Observações vira o Histórico do Atendimento.
  • Use o botão 📋 Histórico Clientes para ver o prontuário completo.

☁️ 5. Sincronização e Cloud

  • O botão 🔄 Sincronizar busca os dados mais recentes da nuvem (Google Sheets).
  • Se aparecer "OFFLINE", verifique sua internet ou tente novamente em alguns segundos.
  • Os dados são salvos automaticamente no navegador também, para consulta offline.
  • Todos os dispositivos conectados visualizam os mesmos dados após a sincronização.

📤 6. Relatórios e Exportação

  • Clique em 📤 Relatórios para gerar listagens.
  • Escolha o período: Hoje, Mês Atual ou Geral.
  • Filtre por sala específica ou todas.
  • Use Gerar Prévia para visualizar antes de baixar.
  • Clique em 📥 Baixar PDF para salvar o relatório em arquivo.
  • Use 📋 Histórico Clientes para exportar o prontuário do paciente.

🖨️ 7. Impressão

  • Use o botão 🖨️ Imprimir no topo para imprimir a tela atual.
  • Recomendado usar no navegador Chrome para melhor formatação da grade.

🔧 8. Administração (Acesso Restrito)

  • Rateio: Visualize horas e faturamento por profissional.
  • Dashboard: Estatísticas de ocupação, clientes, faltas e uso.
  • Auditoria: Log completo de criações e exclusões.
  • Config. Salas: Cadastre, edite ou remova salas e valores por hora.
  • Backup: Exporte todos os dados em JSON ou restaure de um arquivo anterior.

💡 Dicas Rápidas

  • 🌓 Use o botão de tema para alternar entre modo claro e escuro.
  • 🔧 O botão de ferramentas ativa o painel de diagnóstico técnico (para suporte).
  • Se algo não carregar, pressione F5 ou clique em Sincronizar.

📱 9. WhatsApp - Notificações

O módulo WhatsApp permite enviar confirmações e lembretes de consulta diretamente pelo WhatsApp:

  • Clique em 📱 WhatsApp no topo para abrir a Central de WhatsApp.
  • Configure o nome da clínica, endereço e assinatura uma única vez.
  • Cadastre o celular de cada profissional na aba 📱 WhatsApp do painel Admin.
  • Ao criar uma reserva, marque as opções de notificação (paciente e/ou profissional).
  • O botão 📱 em cada slot da grade permite reenviar a confirmação a qualquer momento.
  • Use 📅 Notificar Pacientes de Hoje para enviar lembretes em massa para o dia atual.
  • Na Vista Lista, o botão 📱 em cada linha abre o envio para aquela reserva.
  • O sistema usa links wa.me — o WhatsApp abre em nova aba com a mensagem preenchida.
  • Personalize o texto das mensagens em Admin > Aba WhatsApp. Use variáveis como {PACIENTE}, {DATA}, {HORA_INICIO} etc.
  • Os números dos pacientes são lidos automaticamente do cadastro de clientes (campo celular, formato 000-00000-0000).

⚠️ Como funciona o WhatsApp

O FisioSalas utiliza o WhatsApp Web / API não oficial via links wa.me. Ao clicar em enviar, o WhatsApp abrirá em nova aba com a mensagem preenchida — basta confirmar o envio.

Para envio automático em massa, considere integrar com a API oficial do WhatsApp Business. Os números dos pacientes são lidos do cadastro de clientes (campo celular).
